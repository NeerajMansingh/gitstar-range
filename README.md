This repository gives most stared repository according to given date range. It is a CLI tool.

To use this as a command, run this in terminal :-

npm install -g

Then it can be used by typing  => findstarredgithubrepo

Maximum number of requests allowed are 10/min. Please wait for a minuite if this limit is exceeded.

Example: (can be use byfindstarredgithubrepo if above instructions are followed )

$ ./index.js
